Question-1: What are the different ways to select an element in the DOM?
There are few different ways to select element in the DOM:
* document.getElementById; Finds any elements by its unique ID.
* document.getElementsByClassName; Selects elements by their ClassName .
* document.getElementsByTagName; Selects elements by their HTML tag
Question-2: What is the difference between innerHTML, innerText, and textContent ?
* innerHTML: Shows all content inside an element, including HTML tags.
* innerText: Displays only the visible text,it mainly fucuses on what is actually displayed on the screen.
* textContent: Shows every bit of text, including hidden part of the code.
Question-3: What is event delegation in the DOM?
Event delegation is like having a smart event listener! Instead of adding listeners to every button or link, it attach one to a parent element (like a container). When a child is clicked, the event "bubbles up" to the parent, so we can check which child triggered it. Saves time and works for dynamic content!
Question-4: What is event bubbling in the DOM?
Event bubbling means when an event happens on an element, it first runs handlers on that element, then on its parent.
Question-5: How do you create, add, and remove elements using JavaScript?
* To create a new element, we use document.createElement()
* To add it, we can use parentElement.appendChild()
